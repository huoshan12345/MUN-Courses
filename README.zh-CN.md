# MUN-Courses（Markdown 版本）

[English](README.md) | [中文](README.zh-CN.md)

> 非官方个人学习笔记。

这个仓库收录了我将老师原本以 PDF 形式发布的课件和作业，重写为 Markdown 的版本。

## 为什么做这个仓库

PDF 里的文字通常不方便复制和复用。转换为 Markdown 后，内容更容易复制、编辑和引用，尤其适合在 AI 辅助学习场景中使用。

## 仓库结构

- `ENGI-9867(Adv Comp Concepts for Engi)/`：课程周内容与作业的 Markdown 重写版本
- `symbols.md`：常用符号集合，方便直接复制粘贴

## 关于图片和图示

Markdown 不太适合完整复现原课件中的所有视觉内容，因此部分图片或图示会转换为文字形式表达。

例如：
- 状态转换图可能会改写为状态转换表达式。

## 准确性与显示说明

在重写过程中，可能会出现少量输入错误或显示问题（例如在 GitHub 网页端或某些 Markdown 预览器中）。

如果你发现问题，欢迎提交 issue 或 PR。
